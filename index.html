<http-cat> web component

This is how you embed a HTTP Cat using web components (with Polymer):

<http-cat status="404"></http-cat>

A http-cat web component looks like this:

API

Available attributes

  • status - The HTTP status code to show the related cat image
  • container - Selector with the element where the img element will be inserted. If not provided, the image will be appended after the http-cat element
